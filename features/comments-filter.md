# Comments Filter

The **Comments Filter** adds a filter panel to Polymarket's comment sections — letting you filter comments by outcome and position to cut through noise.

---

## What It Does

PolyHelper injects a filter bar above the comments section. You can filter by:

| Filter | Options |
|---|---|
| **Trader** | All traders / specific tracked traders from your Trader Tracker |
| **Market** | All Markets / specific sub-market |
| **Outcome** | All Outcomes / YES / NO |
| **Min position size** | Custom $ amount threshold |

When you select a specific outcome, only comments from traders holding that position are shown.

---

## Integration With Trader Tracker

If you have traders added to [Trader Tracker](trader-tracker.md), their names appear in the Trader dropdown — letting you quickly see only their comments across any market.

---

## Markets Where This Feature Activates

Works on all Polymarket event pages with a comment section.
