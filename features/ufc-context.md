# UFC Context

The **UFC Context** panel loads fighter profiles, records, and rankings directly inside Polymarket — giving you everything you need to trade UFC and MMA markets without leaving the page.

![UFC Context Panel](https://raw.githubusercontent.com/polymarkethelper/gitbook/main/.gitbook/assets/ufc-context.png)

---

## What It Shows

### Fighter Profiles
For each fighter involved in the market:
- Full name and current UFC ranking (divisional)
- Physical stats: height, reach, weight class
- Professional MMA record: Wins / Losses / Draws
- Win methods breakdown: KO/TKO, Submission, Decision
- Recent fight history: last fights with results, opponents, and dates

### Rankings
- Current divisional ranking
- Pound-for-pound ranking where applicable
- Champion and top contenders for context

---

## Data Source

Fighter data comes from [ufc.com](https://www.ufc.com).
